- 登录/注册页新增“登录/注册 Aivro，即代表同意隐私政策和服务条款”提示，链接到对应页面；登录/注册切换和输入区域新增过渡动画。
- 登录页提交、注册邮箱验证码发送、`/image` 生图按钮和生成中结果卡片改为 Anime.js `createDrawable` 的 Aivro 线稿加载动画，支持浅色/深色主题。
- 管理后台系统设置新增“云存储”页签，支持 Cloudflare R2 / S3、Endpoint、Region、Access Key、Bucket、Public Base URL、图片/视频路径模板、图片/视频过期天数、自动清理和 Path Style Endpoint 配置，并提供测试连接按钮；Secret Access Key 留空表示沿用已保存密钥。
- 新增 `cloud_files` 表记录云端生成文件；开启云存储后图片生成和视频生成会由后端转存到 R2/S3，前端展示、预览、保存素材、下载和生成历史使用云端 `public_url`，不再把生成结果写入浏览器本地图片/视频存储。
- 新增 `generation_histories` 表，生图工作台和视频创作台的生成历史改为保存到数据库；历史记录只保存已关联 `cloud_files` 的云端图片/视频，到期、删除或媒体不存在时列表接口会自动移除对应历史。
- 用户侧 API 配置入口已删除，前端不再允许用户填写 Base URL、API Key 或选择本地直连；图片、视频、文本和画布生成均只使用后台管理员配置并公开开放的模型。
- 云存储图片和视频分别按各自配置写入 `expires_at`，默认均为 7 天；自动清理任务会删除到期且未删除的云端对象并标记 `deleted_at`。
- 新增基础 i18n 能力，支持 `zh-CN` 和 `en-US`；首次进入按浏览器语言选择，手动切换后写入本地缓存并优先使用缓存语言，新增云存储文案提供中英文翻译。
- 顶部语言按钮改为下拉菜单，可选择中文和 English；切换后会同步 Ant Design 组件语言、页面 `lang`、顶部导航、移动端导航和首页主要文案。
- 顶部导航去掉版本按钮和 GitHub 按钮，用户侧“我的画布”入口文案改为“工作流”，工作流列表、空状态、创建、导入、删除和清空等主要操作文案同步调整。
- 管理后台系统设置新增“邮件配置”页签，支持配置 SMTP 验证码、验证码有效期、注册绑定邮箱、找回密码和 MetaMask 邮箱验证邮件模板，并在后台提供模板预览。
- 管理后台邮件配置改为左右分栏：左侧配置 SMTP 并支持填写邮箱测试发送，右侧模板以块展示，点击后在大弹窗中左侧编辑模板、右侧实时预览；邮件模板新增 `{{ip}}`、`{{country}}`、`{{region}}` 变量。
- 管理后台新增“数据库配置”独立页面，可查看数据库结构是否完整、执行来源文件、缺失模型和每次更新记录；系统设置页不再放数据库更新按钮。
- 公开配置新增“是否开启邮箱验证”，开启后账号密码注册需要填写邮箱和验证码；新增找回密码页面，通过邮箱验证码重置密码。
- 管理后台系统设置新增“第三方登录”页签，Linux.do 登录迁移到该页签，并新增 Google、GitHub、MetaMask 和自定义 OAuth 登录配置，默认保留 O2 自定义登录项。
- 管理后台第三方登录配置为 Google、GitHub、MetaMask 补充登录图标，和 Linux.do 的图标展示保持一致。
- 管理后台第三方登录改为卡片列表，卡片展示开启状态和使用人数，点击后在弹窗中编辑；新增自定义第三方登录会创建新卡片。
- Docker 编排移除 PostgreSQL 的宿主机端口映射，数据库只允许应用容器通过内部网络访问。
- 登录页新增 Google、GitHub、Linux.do、自定义 OAuth 和 MetaMask 登录入口；MetaMask 签名后进入单独邮箱验证页面，验证后记录钱包地址并完成登录。
- 用户管理适配第三方登录信息，展示登录来源、第三方 ID、MetaMask 钱包地址和邮箱验证状态；第三方头像会作为用户头像保存并展示。
- 修复删除画布图片节点或清空画布后撤销时，节点信息恢复但本地图片数据已被清理导致图片丢失的问题。
- “我的素材”类型筛选区右侧新增文本样式的导出素材和导入素材入口，可将全部素材导出为包含 `assets.json` 与图片、视频文件的压缩包，并从压缩包恢复素材。
- 未登录状态下，画布右上角不再显示用户头像菜单、用户名称、算力点余额和退出登录入口，改为显示登录入口；快捷键入口仍可直接打开。
- 生图工作台的图片参数区复用画布里的紧凑版图像设置面板，尺寸、质量、生成张数的交互保持一致；工作台仍保留独立的模型选择。
- 生图工作台生成记录改为数据库持久化：每次生成会保存提示词、参考图、模型、质量、尺寸、张数和云端图片信息；点击历史记录会回填本次生成配置并预览结果。
- 视频设置抽成画布和视频创作台共用的紧凑面板，清晰度、尺寸、秒数按固定网格选择并支持手动输入；尺寸选择 `auto` 时请求不传 `size`。
- 修复画布和生图工作台选择图片尺寸后，请求图片生成/编辑接口未携带 `size` 参数的问题；`auto` 不传，其余比例或像素尺寸会随请求发送。
- 修复生图工作台和画布生图请求中 `quality` 参数可能传入上游不支持值导致 400 的问题；请求前会归一化质量枚举，`auto` 或异常值不再发送给上游。
- 管理后台新增/编辑渠道时，渠道可用模型支持通过弹窗按“新获取、已有”分组选择，并可在弹窗内手动增加模型或拉取模型列表后再写回表单。
- 管理后台编辑渠道时，API Key 留空不再触发必填校验，表示沿用已保存的密钥；新增渠道仍要求填写 API Key。
- 管理后台公开配置里的系统可用模型候选项改为由已启用渠道中选择的模型合并去重生成，最终开放哪些模型仍由公开配置里手动勾选。
- 视频生成请求参数对齐 `grok-imagine-video` 接口：使用 `resolution_name`、`preset=normal`、`input_reference[]`，支持清晰度、尺寸、秒数快捷选择和手动输入，并支持最多 7 张参考图。
- 画布视频设置浮层改为挂载到页面根层级并使用自建浮层交互，避免被节点悬浮工具栏遮挡或点击面板内容时关闭。
- 画布生成配置节点的生图参数改为复用图像设置浮层，支持在同一个入口里调整质量、尺寸和生成张数。
- 视频清晰度输入框改为只输入数字，提交请求时再拼接 `p` 单位。
- 视频生成前端会识别后端 `{ code, msg }` 错误响应，创建失败不再继续轮询 `undefined`。
- 新增 `/video` 视频创作台页面，参考生图工作台布局，支持提示词、参考图、视频参数、生成结果、保存素材、下载和数据库生成记录；清晰度和秒数均支持常用值选择与手动输入，生成记录保存云端媒体信息并可回填本次提示词、参考图和参数。
- 视频创作台生成前会把模型、尺寸、秒数、清晰度归一化为视频接口支持的参数，并展示后端返回的错误信息，避免页面侧残留的生图参数影响视频请求。
